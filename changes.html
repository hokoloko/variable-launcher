Изменения в API

  1. исправлен баг связанный с тем, что бинды кастомных модулей не вызызвали onToggleListener и onClickListener.
  2. исправлен метод ModuleManager.addModules - не добавлял модули.
  3. добавлен метод LocalPlayer.setMoveButtonStatus(int id, boolean status);
  4. метод LocalPlayer.setPositionXRelative переименован в LocalPlayer.setPositionRelativeX.
  5. метод LocalPlayer.setPositionYRelative переименован в LocalPlayer.setPositionRelativeY.
  6. метод LocalPlayer.setPositionZRelative переименован в LocalPlayer.setPositionRelativeZ.
  7. добавлен константный класс MoveButton - нужен для методов LocalPlayer.isMoveButtonPressed и LocalPlayer.setMoveButtonStatus.
  8. изменен метод Inventory.setArmor. Теперь он требует 3 аргумента: slot - слот с броней, armorSlot - слот брони (от 0 до 3), freeSlot - пустой слот на случай если уже надета броня.
